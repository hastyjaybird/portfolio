# Portfolio Website

Next.js portfolio for Dr. Julia Hasty.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Recommended: [Vercel](https://vercel.com)

```bash
npx vercel
```

## Content

Site copy is driven by `src/lib/content.ts`. Update that file or sync from `../content/master-profile.md` when the master profile changes.
